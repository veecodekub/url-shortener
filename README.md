# url-shortener

To install dependencies:

```bash
bun install
```

To run development server:

```bash
bun dev
```

To docker build:

```bash
docker build --pull -t url-shortener .
```

To run docker container:

```bash
docker run -d -p 8080:8080 url-shortener
```

The application will be accessible at `http://localhost:8080`.
